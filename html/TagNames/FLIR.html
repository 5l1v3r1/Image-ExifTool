FLIR RawData Tags

The thermal data may be stored either as raw data, or in PNG format. If stored as raw data, ExifTool adds a TIFF header to allow the data to be viewed as a TIFF image. If stored in PNG format, the PNG image is extracted as-is. Note that most FLIR cameras using the PNG format seem to write the 16-bit raw image data in the wrong byte order.
