GPS Tags

ExifTool is very flexible about the input format for lat/long coordinates, and will accept 3 floating point numbers separated by just about anything. Many other GPS tags have values which are fixed-length strings. For these, the indicated string lengths include a null terminator which is added automatically by ExifTool.
